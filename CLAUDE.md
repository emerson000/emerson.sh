# Claude Code Instructions

## Git Conventions

- **Commit messages**: Use [Conventional Commits](https://www.conventionalcommits.org/) — `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`, `style:`, `test:`, etc.
- **No attributions**: Do not include Claude session URLs or any Claude attribution in commit messages, PR titles, PR bodies, or code comments.
- **Branch naming**: Always use a type prefix — `feat/`, `fix/`, `chore/`, `docs/`, `refactor/`, etc.
