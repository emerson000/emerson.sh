This is a modern portfolio website built with `Next.js` and MDX. It features:

- Dynamic content types (blog posts, projects, etc.)
- MDX support for rich content
- Modern UI with Tailwind CSS
- Responsive design

## Technical Stack

- Next.js 14
- TypeScript
- MDX
- Tailwind CSS
- Shadcn UI

## Features

- Dynamic routing based on content types
- MDX support for rich content
- Modern and responsive design
- Dark mode support
- Syntax highlighting for code blocks
